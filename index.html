What is Airtab Remote and who is it for?

Airtab remote allows you to control your Media PC over a LAN. If you're like us, you're on your laptop or tablet all the time, even when sitting in front of a TV, and if you also have a media PC, why use another keyboard when you could just open a tab in your browser and use the keyboard, mouse or touchpad you already have in front of you?

Getting Started (easy)

  1. Download our installer and install it on your Windows media PC
  2. Right click on on the Airtab server icon on your desktop and select "run as Administrator". The server uses Nodejs, so allow that process to listen on port 8090 on your firewall if you have a firewall running.
  3. On another computer, open browser and hit the IP of your media PC on port 8090, e.g. http://192.168.1.5:8090
  4. Put your wireless keyboard and mouse away.

I'm a developer and I want to help

That's great! We're happy to accept contributions, just fork it on github. Airtab was built using nodejs, socket.io and C# .Net.

It doesn't work :(

This is an early beta version of Airtab. We've only tested it on Windows 7 as a server because that's what we use. Let us know by creating an issue on github for bug reports or feature requests.

Does it work on my Mac or Linux computer?

The server (media PC component) only works on Windows 7 or higher at the moment, but the client can be run from anywhere (we Mac OSX with Chrome most of the time). We recommend Chrome since we've made a chrome extension for convenience.

Known issues

  1. Unless you run the server as an Administrator, Airtab will not work when you're performing administrative tasks, like using the task manager or installing programs.
  2. It doesn't work for the log in screen, not even when it's running as Administrator. Windows has a log of security that makes this really hard, and we haven't figured out how to get around that yet.
